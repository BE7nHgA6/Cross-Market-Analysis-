# Cross-Market-Analysis-
# 📊 Cross Market Analysis Dashboard

This project is a **data-driven dashboard** built using **Python, SQL, and Streamlit** to analyze relationships between different financial markets — including **Cryptocurrency, Oil, and Stock Markets**.

## 🚀 Features

- 📊 **Market Overview**
  - Displays average prices of Bitcoin, Oil, S&P 500, and NIFTY
  - Interactive date range selection
  - Combined market snapshot table

- 🔍 **SQL Query Runner**
  - Predefined SQL queries for real-time analysis
  - Covers Crypto, Oil, Stocks, and Cross-market joins
  - Helps explore trends, volatility, and correlations

- 💰 **Top 3 Crypto Analysis**
  - Focused analysis on top cryptocurrencies by market cap
  - Interactive price trend visualization
  - Custom date filtering

## 🛠️ Tech Stack

- **Python**
- **Pandas** – Data processing
- **SQLite** – Database storage
- **Streamlit** – Interactive dashboard
- **yFinance API** – Stock data
- **GitHub Dataset** – Oil price data

## 📂 Data Sources

- Cryptocurrency data (market + price)
- Oil price dataset (WTI daily prices)
- Stock market data (S&P 500, NASDAQ, NIFTY)

## 📈 Key Insights

- Compare Bitcoin with Oil and Stock indices
- Analyze market trends and volatility
- Perform cross-market correlation analysis

## 🎯 Objective

To build an **interactive financial analytics platform** that demonstrates:
- SQL query handling
- Data integration from multiple sources
- Real-time dashboard visualization
